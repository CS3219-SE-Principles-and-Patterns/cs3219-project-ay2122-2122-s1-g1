# cs3219-project-ay2122-2122-s1-g1

## Client

In `/client` folder,
- Run `npm install`
- Run `npm start`
- Access `http://localhost:3000`

## Server (port 5000)

In `/server` folder,
- Run `npm install`
- Run `npm start` or `npm run devStart` (runs with nodemon)
- Backend endpoints can be accessed at `http://localhost:5000`
